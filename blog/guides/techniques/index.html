Techniques
astrologiques

Comment lire un thème natal dans le temps. Révolution solaire, progressions secondaires, profections annuelles — les outils pour comprendre ce qui se passe maintenant dans ta vie.

Le thème natal est une carte fixe — mais la vie bouge. L'astrologie prédictive utilise plusieurs techniques pour lire comment les énergies natales se déploient dans le temps : certaines regardent l'année en cours, d'autres les cycles de vie sur plusieurs décennies. Ces techniques se lisent toujours en dialogue avec le thème natal — elles ne créent pas de nouveaux événements, elles activent ce qui est déjà là.
